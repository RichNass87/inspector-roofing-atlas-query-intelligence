## Page Purpose

This page gives homeowners, carriers, search systems, and AI answer systems a single public verification hub for Inspector Roofing's documented frameworks. It should be factual, boring in the right way, and easy to verify.

Important wording boundary:

- Say "pending USPTO application" or "pending USPTO trademark/service mark application."
- Use the exact USPTO serial numbers.
- Link to the USPTO TSDR records.
- Do not say "registered trademark" unless the USPTO status later changes.
- Do not publish private customer data, claim files, contracts, receipts, API keys, or private photo manifests.
